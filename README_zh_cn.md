# 牧场之趣 (Enjoyable Animal Husbanding)

一款为 Minecraft 1.20.1 Forge 开发的模组，为游戏中的动物添加性别机制与更真实的繁殖逻辑。

作者：**brodong**

## 已实现功能

### 性别系统
- 每只动物（`Animal` 实体）在生成时随机获得 **雄性（Male）** 或 **雌性（Female）** 性别。
- 性别数据通过 NBT 持久化存储，重新加载存档后性别保持不变。
- 只有异性动物之间才能进行繁殖，同性别动物交配会被阻止。

### 繁殖 AI 过滤
- 动物的繁殖 AI（`BreedGoal`）只会寻找异性伴侣，不会浪费时间靠近同性别目标。
- 与 `canMate()` 拦截形成双重保障，确保性别检查不会被绕过。

### 检查棒
- 新道具：**检查棒**。
- 手持检查棒右键点击任意动物，即可在聊天栏查看该动物的性别（雄性/雌性/未知）。
- 合成配方：**1 木棍 + 1 钻石**（无序合成，工作台任意位置均可）。
- 在创造模式中可在 **工具** 一栏找到。

## 安装

1. 安装 **Minecraft Forge 1.20.1**（版本 47.x 及以上）。
2. 从 [Releases](https://github.com/brodong/enjoyable-animal-husbanding/releases) 下载模组 `.jar` 文件。
3. 将 `.jar` 文件放入 Minecraft 的 `mods/` 文件夹。
4. 使用 Forge 配置文件启动游戏。

### 从源码构建

```bash
git clone https://github.com/brodong/enjoyable-animal-husbanding.git
cd enjoyable-animal-husbanding
./gradlew build
```

编译好的 jar 位于 `build/libs/` 目录下。

## 使用说明

### 繁殖
- 用对应的繁殖物品（如小麦喂牛）喂养两只**异性**动物即可。
- 同性别动物无法繁殖。如果不确定动物性别，可使用检查棒右键查看。

### 检查棒
- 合成：**1 木棍 + 1 钻石**（工作台任意位置）。
- 手持检查棒**右键**动物，聊天栏将显示"xxx 的性别为：雄性/雌性"。

### 配置文件
配置文件位于 `config/enjoyable_animal_husbanding-common.toml`，目前包含示例调试选项。
